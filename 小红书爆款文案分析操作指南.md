## 🚀 部署步骤

### 步骤1: 等待下载完成
文件正在自动下载中，预计还需要几分钟。

### 步骤2: 解压文件
```bash
# 进入目录
cd /Users/imac/.openclaw/workspace/xiaohongshu-mcp

# 解压文件
tar xzf xiaohongshu-mcp-darwin-arm64.tar.gz

# 检查解压后的文件
ls -la
```

应该看到以下文件:
- `xiaohongshu-login-darwin-arm64` (登录工具)
- `xiaohongshu-mcp-darwin-arm64` (MCP服务器)

### 步骤3: 设置执行权限
```bash
chmod +x xiaohongshu-login-darwin-arm64
chmod +x xiaohongshu-mcp-darwin-arm64
```

### 步骤4: 登录小红书账号
```bash
# 运行登录工具
./xiaohongshu-login-darwin-arm64
```

**登录流程**:
1. 程序会自动打开浏览器窗口
2. 使用小红书账号扫码登录
3. 登录成功后自动保存cookies
4. 关闭浏览器窗口

**注意事项**:
- 不要同时在多个网页端登录同一账号
- 可以使用手机App查看账号状态
- cookies保存在 `~/.xiaohongshu/cookies.json`

### 步骤5: 启动MCP服务器
```bash
# 启动服务器（无头模式）
./xiaohongshu-mcp-darwin-arm64

# 或者启动有界面模式（便于调试）
./xiaohongshu-mcp-darwin-arm64 -headless=false
```

**服务器信息**:
- 地址: `http://localhost:18060/mcp`
- 端口: `18060`
- 首次运行会自动下载浏览器（约150MB）

### 步骤6: 验证连接
```bash
# 测试MCP连接
curl -X POST http://localhost:18060/mcp \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"initialize","params":{},"id":1}'
```

## 📱 小红书运营建议

### 1. 内容策略
- **发布时间**: 18:00-22:00用户活跃高峰
- **标题长度**: 不超过20个字
- **正文长度**: 不超过1000个字
- **图片数量**: 3-9张最佳
- **标签使用**: 添加相关标签增加曝光

### 2. 爆款公式
```
爆款 = 情感共鸣 + 实用价值 + 视觉冲击 + 互动引导
```

### 3. 标题模板
```
数字吸引: "3个技巧让你..."
情感钩子: "破防了！这..."
利益承诺: "让你轻松..."
悬念设置: "没想到..."
```

### 4. 内容结构
```
1. 痛点引入: 描述用户问题
2. 解决方案: 具体方法步骤
3. 效果展示: 前后对比/成果
4. 价值升华: 意义/感悟
5. 互动引导: 提问/求关注
```
