## 📊 爆款文案特征分析

### 标题技巧（必学！）
1. **数字具体化**：3天、5个步骤、7天计划
2. **情绪共鸣**：终于、竟然、哭死、绝了
3. **利益明确**：省钱、变美、涨知识、省时间
4. **悬念设置**：但是、竟然、没想到、慎入

### 内容结构（黄金公式）
```
开头（3秒抓人）：痛点/共鸣/悬念
中间（干货密集）：步骤/方法/经验
结尾（引发互动）：提问/征集/福利
```

### 视觉要点
- 封面图：清晰、有重点、色彩鲜明
- 内页图：步骤清晰、前后对比
- 视频：前3秒必须吸引人
